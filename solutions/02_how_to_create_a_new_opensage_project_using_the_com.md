# How to create a new OpenSage project using the command line interface

_Use the OpenSage CLI to generate a new project template_

## Steps

1. Open a terminal and navigate to the desired project directory
2. Run the command `opensage init myproject` to create a new project template
3. Follow the prompts to configure the project settings
4. Verify the project structure by running `tree myproject`
5. Start the development server with `opensage serve`

## Pitfalls

- Not navigating to the correct directory before running the init command
