# How to train an OpenSage agent using a custom dataset

_Prepare and load the dataset, then train the agent using the OpenSage API_

## Steps

1. Prepare the dataset by formatting it according to the OpenSage documentation
2. Load the dataset into OpenSage using the `opensage load_data` command
3. Configure the agent settings using the `opensage config` command
4. Train the agent using the `opensage train` command
5. Evaluate the agent's performance using the `opensage evaluate` command

## Pitfalls

- Not formatting the dataset correctly before loading it into OpenSage
